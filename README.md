# Kidney Exchange Optimization

This project applies integer programming methods to model and optimize kidney exchange systems, where donor–recipient pairs are matched to maximize the number of successful transplants.

## Overview
- Developed optimization models using AMPL and MATLAB  
- Implemented multiple formulations to evaluate feasibility and performance  
- Worked with structured datasets to simulate real-world matching scenarios  

## Methods
- Integer programming models for donor–recipient matching  
- Feasibility testing and relaxation techniques  
- Comparative analysis across model formulations  

## Results
The analysis demonstrates how different modeling approaches affect both computational efficiency and the percentage of matched patient–donor pairs. Results highlight key trade-offs between scalability and solution quality as dataset size increases.

## Project Structure
- **Final_Writeup.pdf** – full methodology and analysis  
- **visualizations/** – model outputs and performance graphs  
- **code/** – core optimization models (AMPL/MATLAB)  

## Reproducing the Model (Optional)
This project uses AMPL and MATLAB for optimization and visualization. To run the models, load the `.mod` and corresponding `.run` files in AMPL and execute using the provided datasets.
